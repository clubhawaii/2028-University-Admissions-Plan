학교 로고 폴더
==============

이 폴더에 학교 로고 이미지를 넣으면, 의대 검색기의 학교 카드(및 모달)
네모박스에 자동으로 표시됩니다. 로고 파일이 없는 학교는 기존처럼
대학명 앞 2글자 텍스트로 표시됩니다.

[파일 이름 규칙]
- 데이터의 "대학 전체명" + .png
- 예) 가천대학교.png, 충남대학교.png, 가톨릭대학교.png

[권장]
- 배경이 투명한 PNG (또는 흰 배경)
- 정사각형에 가깝게, 여백 약간 포함
- 너무 큰 파일은 피하기 (가로세로 약 200~400px 권장)

[정확한 대학명 목록 확인 방법]
- 의대 검색기를 브라우저로 열고 개발자도구(F12) 콘솔에서
  UNIVS.join('\n')  입력 → 전체 대학명 목록이 출력됩니다.
  그 이름 그대로 "이름.png"로 저장하면 됩니다.
